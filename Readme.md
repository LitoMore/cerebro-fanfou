# cerebro-fanfou

A Cerebro plugin for Fanfou

## Usage

1. Config

```
ff config CONSUMER_KEY CONSUMER_SECRET
```

2. Login

```
ff login USERNAME PASSWORD
```

3. Post

```
ff Hello Fanfou
```
